The ``add_integer`` function adds two numbers.

>>> add_integer = __import__('0-add_integer').add_integer

Basic usage
>>> add_integer(1, 2)
3
>>> add_integer(100, -2)
98
>>> add_integer(2)
100

Floats are cast to ints
>>> add_integer(100.3, -2)
98
